# CV

## Vefsíða (staðbundin)

Ég bjó til einfaldan, statískan vef fyrir CV í þessari möppu. Skrárnar sem bætt var við eru:

- `index.html`
- `styles.css`
- `script.js`

Til að keyra síðuna staðbundið getur þú einfaldlega opnað `index.html` í vafra eða notað einfalt HTTP-þjónustuforrit.

Dæmi með Python 3 (í verkefnamöppu):

```bash
python3 -m http.server 8000
# opna http://localhost:8000
```

Ef þú villt að ég sérsniði snið, litaval, eða fylli inn raunverulegt efni (nafn, menntun, reynslu), segðu mér frá því og ég uppfæri skrána.
